# Transient Leadership Opinion Dynamics

This repository collects exploratory code, notebooks, figures, and project documents related to transient leadership in opinion dynamics. The materials currently include reproduction-oriented Python scripts, notebook-based experiments, course/project writeups, and representative generated plots. This is an initial repository organization pass intended to make the project uploadable and easier to navigate without refactoring the underlying work.

## Current Contents

- `src/`: Python source files for reproduction and model experimentation
- `notebooks/`: Jupyter notebooks for edited, transient leadership, and radical-vs-populist explorations
- `docs/`: presentation materials
- `reports/`: project writeups and course report PDFs
- `figures/`: generated plots and output figures kept as representative artifacts
- `data/`: placeholder for future datasets or inputs
- `archive/`: reference papers, templates, and other materials retained conservatively during the first pass

## Repository Status

This repository has only received a light first-pass cleanup for upload. The current structure is meant to preserve all existing work while separating source material, outputs, and archival references into clearer folders.
